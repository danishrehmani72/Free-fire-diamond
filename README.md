# Diamond Rush - Premium Free Fire Diamond Store

Diamond Rush is a high-performance, conversion-optimized web application for selling Free Fire diamonds. It features a modern gaming aesthetic with neon accents, glassmorphism, and smooth animations.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express
- **Deployment**: Ready for Vercel / Cloud Run
- **Animation**: Motion (Framer Motion)
- **Icons**: Lucide React

## Key Features
- 🚀 **Instant Delivery**: Automated order processing system.
- 🛡️ **Legit Top-up**: Official API integration ensuring account safety.
- 📱 **Mobile Optimized**: Seamless experience on all devices.
- 📊 **Admin Panel**: Complete analytics, order management, and user controls.
- 💬 **WhatsApp Support**: Direct integration with support line `03482640090`.
- 💳 **Multiple Payment Methods**: EasyPaisa, JazzCash, Binance Pay, and Cards.

## Getting Started

### 1. Environment Setup
Create a `.env` file in the root and add your configuration (see `.env.example`).

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```

## Folder Structure
- `/src/pages`: All main application pages.
- `/src/components`: Reusable UI components.
- `/src/lib`: Utility functions.
- `/server.ts`: Express backend server and API logic.

## Deployment on Vercel
1. Push your code to a GitHub repository.
2. Connect the repository to Vercel.
3. Vercel will automatically detect the build settings.
4. Add your environment variables in the Vercel dashboard.

## Security
- Sanitized inputs.
- Safe route guards for admin.
- Encrypted data transmission.
- Official API usage for top-ups.
